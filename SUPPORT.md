# Support

Onyx Writer is alpha software. Please keep backups of important OKF bundles and include clear reproduction details when reporting issues.

Use GitHub Issues for:

- Reproducible bugs
- OKF round-trip problems
- Tauri packaging problems
- JSONM design-system import or rendering problems
- Documentation gaps

Please include:

- Onyx Writer version or commit
- Operating system
- Whether the issue happens in the desktop app, browser dev server, or both
- Minimal reproduction steps
- Any relevant OKF/Markdown snippet, with private content removed
