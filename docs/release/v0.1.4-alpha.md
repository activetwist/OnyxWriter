# Onyx Writer 0.1.4 Alpha

Onyx Writer `0.1.4-alpha` extends the public alpha release path with Windows build confidence.

## Highlights

- Verified native Windows ARM64 packaging in a Windows 11 Pro VM.
- Produced Windows ARM64 MSI and NSIS installers locally for smoke testing.
- Added GitHub Actions release-matrix support for Windows x64 packages.
- Documented the Windows build host setup, required toolchain, artifact paths, and signing caveat.

## Release Notes

The Windows ARM64 VM build requires Visual Studio Build Tools plus LLVM/Clang for native Rust dependencies. The local VM intentionally does not store Tauri updater signing private keys, so signed updater assets should continue to be produced by trusted release automation.

The GitHub Actions workflow now includes a Windows x64 job so the public release process can generate the mainstream Windows installer target alongside macOS and Linux artifacts.
